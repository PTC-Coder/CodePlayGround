◆ LlRecvAclData()

uint8_t* LlRecvAclData ( void  )

Receive an ACL data packet.

Returns
Data buffer.

Receive an ACL data packet. If no ACL buffers are available this function returns NULL.

Note
The host must deallocate the buffer by calling WsfMsgFree() and call LlRecvBufCmpl() to update LL accounting.
Returns
Data buffer.

Receive an ACL data packet. This function returns a pointer to an ACL buffer formatted according to [1]. The host must parse the header to determine the connection handle, flags, and length fields. If no ACL buffers are available this function returns NULL.

The host must deallocate the buffer by calling WsfMsgFree() and then calling LlRecvAclDataComplete().
