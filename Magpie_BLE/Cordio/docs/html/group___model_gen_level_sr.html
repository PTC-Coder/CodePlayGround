◆ pStoredStates

mmdlGenLevelState_t* pStoredStates

Pointer to the structure that stores current state and scene data. First value is always the current one. Second value is the target state. Sequential values represent scene values starting with scene index 0 and ending with index :MMDL_NUM_OF_SCENES - 1. Structure will store :MMDL_NUM_OF_SCENES + MMDL_GEN_LEVEL_STATE_CNT states.
