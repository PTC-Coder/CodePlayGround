enum mmdlReturnValues

Mesh models return value (status) enumeration.

Values from 0x00 to 0x7F are shared by all models. Values from 0x80 to 0xFF are unique to and defined by each model.

Enumerator

Operation completed without errors

MMDL_OUT_OF_MEMORY 

Memory allocation failed

MMDL_INVALID_PARAM 

Invalid parameter(s) in request

MMDL_BOUND_NOT_FOUND 

No bound was found in the bind table

MMDL_NO_BOUND_RESOLVER 

The bind has no resolver function

MMDL_INVALID_ELEMENT 

Invalid element event status
