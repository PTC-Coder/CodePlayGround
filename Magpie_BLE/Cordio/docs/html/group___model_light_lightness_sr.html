◆ pStoredStates

mmdlLightLightnessState_t* pStoredStates

Pointer to the structure that stores current state and scene data. First value is always the current one. Second value is the target state. Sequential values represent scene values starting with scene index 0 and ending with index :MMDL_NUM_OF_SCENES - 1. Structure will store :MMDL_NUM_OF_SCENES + 2 states.

◆ remainingTimeMs

uint32_t remainingTimeMs

Time remaining until the current state is replaced with the target state. If set to 0, the target state is ignored. Unit is 1 ms.
